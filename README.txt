Основная программа для выполнения симуляций: alltanks.m

Краткое описание программ: 
alltanks.m - выполняет построение графиков зависимости моментов и ЦМ от времени
angle_dependence.m - показывает влияние углов крена и тангажа
max_abs.m - программа предназначенная для поиска максимального значения по амплитуде в векторе
rotation.m - позволяет находить угол общего наклона аппарата (не участвует в работе основного кода)
tankmodel.m - программа, в которая находит ЦМ для отдельно взятого бака
test.m - позволяет проводить проверку отдельных элементов кода без полного моделирования (не участвует в работе основного кода)
